## Overview

The Analyze API provides deep inspection of files and documents. Unlike Guard (which blocks threats) or Redact (which removes PII), Analyze performs comprehensive analysis and returns detailed findings.

## Request

```bash
POST /api/analyze
```

## Use Cases

- Deep inspection of PDF documents for security issues
- Analyzing pull requests for vulnerabilities
- Repository security audits
- Document compliance checks
- Producing transformed artifacts (e.g., redacted documents)
